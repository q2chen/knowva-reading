## MVPのゴール

ユーザーが読んだ本の「感想や学び」を対話的に言語化し、それに基づいて自分の読書傾向（プロファイル）を蓄積・可視化し、AIが次に読むべき本を推薦する。読書体験を深め、次の読書につなげる好循環を生み出すAIエージェントアプリ。

### コアバリュー（最重要）
- **長期保存が本丸:** 会話ベースで聞き出した情報を数年後も振り返れる
- 「今の自分」と「過去の自分」の対話を可能にする
- AIの進化とともに、過去の記録を再解釈できる拡張性

**読書の経験を「時間とともに変化する思想の履歴」として蓄積できる**
- 同じ本を異なる人生のタイミングで読み返したときの変化を可視化
- AIの進化とともに、過去の読書体験を再解釈できる拡張性
- 対話の完全な記録の永続化により、思想の変遷を追跡可能

## AIエージェントの種類と役割

1. **読書の感想や学び・本のどこが好きなのかの言語化をサポートするエージェント**
   - ユーザーとの対話を通じて、読書体験を深掘りする
   - 音声・テキスト入力の両方に対応
   
2. **ユーザープロファイル抽出エージェント**
   - 対話ログからユーザーの属性・価値観・状況を示すプロファイルを抽出・更新する
   
3. **ユーザープロファイルを基に、適切な本を推薦するエージェント**
   - ユーザープロファイルを統合して推薦を生成する

#### MVP主要要件

| No. | 機能名 | 機能概要 |
| :-: | :--- | :--- |
| 1 | **ユーザー登録・ログイン** | メールアドレスとパスワードで基本的な認証を行う。 |
| 2 | **読書記録の追加** | 読んだ本の「タイトル」を入力する（手動またはAPI検索）。 |
| 3 | **音声入力対応（Speech to Text）** | 読書中や読後に、音声で感想・気づきを記録できる。音声認識APIを使用してテキスト化し、対話履歴や学びとして保存する。リアルタイム文字起こしで確認しながら記録可能。 |
| 4 | **AIとの対話的な振り返り** | 読んだ本について、AIエージェントが質問を投げかけ、ユーザーが答える形式で学びや感想を深堀りする。テキスト入力と音声入力の両方に対応。**対話の全ログは生ログ層に保存される。** |
| 5 | **学びの言語化と記録** | AIとの対話を通じて整理された「気づき」「学び」「好きなポイント」をテキストとして自動保存する。 |
| 6 | **ユーザープロファイルの抽出** | 記録された読書データと対話ログから、ユーザーの属性・価値観・状況を示すプロファイルを自動で抽出・更新する。 |
| 7 | **読書履歴の表示** | 過去に読んだ本の一覧と、それぞれの学びの要約を表示する。 |
| 8 | **AIによる本の推薦** | ユーザープロファイルを基に、ユーザーの現在の状態に基づいて次に読むべき本をAIが3〜5冊提案する。 |
| 9 | **生ログの永続化** | すべての対話ログ・入力データをCloud Storageに保存し、将来的な再解釈・アート化に備える。 |

## Agentが聞き出す情報（詳細）

MVPの対話でAgentが聞き出す情報の詳細です：

**読書体験について聞き出す情報:**
- この本の通算読書回数（1回目、2回目...）
- 読み始めた日
- その時の人生の状況（仕事の状況、プライベートの状況、悩みなど）
- なぜこのタイミングでこの本を読んだか・読もうと思ったきっかけ
- 読書スタイル（通読、拾い読み、再読で特定章のみ、など）
- 読書時の心理状態・気分
- 読書環境（場所、時間帯、状況など）

**ユーザーについて聞き出す情報:**
- 現在のライフステージ（学生、若手社会人、中堅、管理職、経営者など）
- 現在の状況（職業、業界、役職、組織規模など）
- キャリアの遍歴（転職歴、専門性の変遷など）
- 現在直面している課題・悩み・テーマ
- 将来の目標・夢・達成したいこと
- 大切にしている価値観・信条・人生哲学
- 読書をする理由・読書に求めるもの

## ユーザージャーニー

0. **初回プロファイル設定（任意）**
   - 現在のユーザーの情報（悩み、ライフステージ等）を入力しておく
   - 音声でも入力可能
   
1. **読む本を登録**
   - タイトル検索またはISBN入力
   - 読書開始日・読書目的を記録
   
2. **読書中の記録（対話的インプット）**
   - 読書中にユーザーが音声またはテキストで感想・気づきを記録
   - AIエージェントが質問を投げかけ、話を引き出す
   - 対話ログがCloud Storageに保存される
   
3. **読了後の振り返り**
   - 読み終わった後に、人生観の変化や軽い感想を記録
   - AIが対話を通じて学びを深掘り
   
4. **ユーザープロファイルの更新**
   - 対話を通じてAIがユーザーの属性・価値観・状況を示すプロファイルを抽出・更新
   - ユーザープロファイルに変化が生じる
   
5. **次の本の推薦**
   - ユーザープロファイルを基に、AIが次に読むべき本を提案
   - 推薦理由には、プロファイルのどの要素が根拠になったかが表示される

**将来的な展開:**
- テキストのみの出力ではなく、アート化（映像化、漫画化、インタラクティブビジュアル）
- 対話の内容から感情や思考の変化を可視化し、読書体験をビジュアル化
